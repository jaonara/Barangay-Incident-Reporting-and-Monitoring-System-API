# Barangay-Incident-Reporting-and-Monitoring-System-API

## Description

A simple REST API for reporting and managing barangay incidents in Iligan City.

## Features
- Create incident report
- View all reports
- Update status
- Delete report

## Tech Stack
- Node.js
- Express.js

## API Base URL
https://your-render-link.onrender.com
